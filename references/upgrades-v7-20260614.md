# 7项模型升级（2026-06-14）

基于5个GitHub仓库（Football-2026, zgzcw-spider, 2026-world-cup-predictor, worldcut-2026）的全量分析。

## 关键发现

### Sporttery API 可达（更新旧SKILL.md）
旧文档说Sporttery API不可达，但实测 `webapi.sporttery.cn` 可直接curl访问：
- URL: `https://webapi.sporttery.cn/gateway/uniform/football/getMatchListV1.qry?clientCode=3001`
- Headers: User-Agent=Mozilla, Referer=https://www.sporttery.cn/, Accept=application/json
- 返回22场赔率（截至2026-06-13），无反爬

### 泊松模型 vs 竞彩赔率的套利机会
用泊松模型概率 vs Sporttery竞彩赔率计算EV，发现3场正EV：
- 巴西主胜 EV+77%, Kelly 9.9% — 泊松60% vs 竞彩2.95(隐含34%)
- 苏格兰客胜 EV+32%, Kelly 7.8% — 泊松65% vs 竞彩2.03(隐含49%)
- 土耳其客胜 EV+71%, Kelly 9.4% — 泊松59% vs 竞彩2.90(隐含34%)

### 竞彩官方 vs 17家公司亚盘的方向矛盾检测
卡塔尔vs瑞士：竞彩赔率暗示主胜+37%vs客胜36%（几乎均势），但17家公司亚洲盘口均值为受1.85球（瑞士让近2球）— 严重矛盾。

## 升级1: Kelly仓位管理 (`kelly_engine.py`)
- Kelly公式: `f = (b*p - q) / b`, 默认1/4分数凯利
- 五级价值分类: edge>=8%高价值, >=4%中高, >=1%中, >=-2%中性, <-2%低价值
- 分级仓位: 不投(0%), 轻仓(<5%), 中仓(5-15%), 重仓(15-25%), 超重仓(>25%)

## 升级2: 多公司赔率聚合 (`multi_bookmaker_engine.py`)
- Sporttery官方赔率（竞彩）去水概率
- 500.com 17家公司亚盘均值/标准差
- 竞彩vs亚盘方向矛盾检测
- 盘口映射: 受球半/两球→-1.75, 半球→0.5, 一球→1.0 等

## 升级3: 盘口异动检测 (`odds_movement_engine.py`)
- 水位变化方向检测（↑升↓降）
- 水位变化≥0.06触发信号
- 强/中强度分级（≥0.12强）
- 资金流向判断（主水降→资金流向主队）

## 升级4: 赔率快照 (`odds_snapshot_cron.py`)
- 定时抓取Sporttery赔率
- 保存到odds_snapshots表
- 配合盘口异动检测使用

## 升级5: 年龄-峰值模型 (`age_peak_model.py`)
- 球员年龄→峰值系数映射（22岁以下0.3→25-28岁黄金期0.9→34岁以上老化）
- 位置权重（守门员1.0, 前锋1.5）
- 大赛经验指数（caps/50 + 首次参赛打折）
- 通过Wikipedia阵容数据计算球队年龄结构

## 升级6: 热门陷阱检测 (`hot_trap_detector.py`)
- Elo极高+排名前3→盛极而衰
- 身价虚高但排名一般→溢价提示
- 高位球队自动标记

## 升级7: 规则化复盘系统 (`settlement_review.py`)
- 赛后自动验证预测命中
- 模型校准（按置信度区间统计准确率）
- model_calibration表写入

## 柜台票单格式（已验证）
当用户要求详细投注票表时，用此排版：
```
┌──────┬──────────────────────┬────────┬───────┬──────────┬─────────┐
│ 序号 │ 选项                 │ 玩法    │ 赔率   │ 金额     │ 预期回收│
├──────┼──────────────────────┼────────┼───────┼──────────┼─────────┤
│    1 │ 瑞士胜                │ 胜平负 │ 2.43  │  30元    │  72.9元 │
│    2 │ 让球平(瑞士让2球)     │ 让球胜 │ 6.40  │  15元    │  96.0元 │
│      │                      │ 平负   │       │          │         │
└──────┴──────────────────────┴────────┴───────┴──────────┴─────────┘
```
关键规则：
- 选项必须写完整全称（"瑞士胜"而非"负"，"让球平(瑞士让2球)"而非"让平"）
- 350元方案用三层架构：A正EV主力60% + B负EV娱乐20% + C让球博冷+平局防守20%
- 按比赛分组显示，每场下方汇总小计，最后总账汇总

## Sporttery API 成功验证
- Endpoint: `https://webapi.sporttery.cn/gateway/uniform/football/getMatchListV1.qry?clientCode=3001`
- 返回格式: JSON, value.matchInfoList[].subMatchList[].oddsList[poolCode=HHAD/HHAF/CRS/TWS]
- HHAD: h=主胜, d=平局, a=客胜
- HHAF: h=让球主胜, d=让球平, a=让球负, goalLine=让球数
- 返回22场世界杯比赛赔率（2026-06-13测试）
