## Elo差与准确率关系

| Elo差区间 | 准确率 | 说明 |
|:|:|:|:|
| [-500, -100) | 42.1% | 弱队冷门多 |
| [-100, 0) | 75.0% | 接近实力，预测准 |
| [0, +100) | 36.4% | 微弱优势，预测最差 |
| **[+100, +300)** | **70.8%** | **明显优势，最准区间** |
| [+300, +600) | 50.0% | 样本少 |

## 关键Pitfall

1. **平局预测=0是结构性缺陷**：标准Poisson模型中p_D永远不是最高概率，需要GBM补充
2. **欧洲队被低估**：Elo无法捕捉阵容深度+大赛经验，需要市场赔率校准
3. **小组赛冷门多**：54.2%准确率，淘汰赛56.6%
4. **Elo差0-100预测最差**：实力接近时区分能力不足
5. **环境修正影响小（2022）大（2026）**：2022卡塔尔空调场馆，2026有墨西哥城2240m+迈阿密高温
