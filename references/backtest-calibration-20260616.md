## 核心发现

### 1. Sporttery 统一前水率 12.9-13.0%
所有场次统一前水率，导致所有选项 EV=-11%。Sporttery 赔率不适合直接作为概率估计。

### 2. 平局率被严重低估
- 实际平局率: **44%** (7/16)
- Sporttery 隐含平局概率: 仅 21-28%
- 平局概率修正建议: Sporttery隐含 +15%

### 3. 热门方向偏差严重
Sporttery 去水概率最高的选项（预测方向）仅 4/12 正确(33%)。
典型: 伊朗胜去水57% → 实际2-2平局

### 4. 模拟投注结果
- 12场 x 100元 = 投入1200元
- 假设每次都买去水概率最高的选项
- 回报: 763元
- ROI: **-36.4%**

## 校准建议

| 问题 | 建议 |
|:----|:------|
| Sporttery 前水率过高 | 引入多公司欧赔聚合，Sporttery 只作为市场情绪指标 |
| 平局低估 | 平局概率修正: Sporttery隐含 +15%，让球盘<0.5时平局乘1.2 |
| 热门偏差 | 引入 Elo 差值做交叉验证，Elo 差<150时下调高分方向权重 |
| 单一源不可靠 | 需要 FootyStats/WhoScored 基本面数据做独立概率估计 |
