# 竞彩票据分析指南

> 场景：用户发来竞彩投注票据截图，需要分析合理性并给出优化建议

## 竞彩六种玩法

| 玩法 | 代码 | 说明 | 典型赔率范围 |
|:|:|:|:|
| SPF | 胜平负 | 标准1X2 | 1.10~15.00 |
| RQSPF | 让球胜平负 | 加让球后1X2 | 1.30~8.00 |
| CRS | 比分 | 精确比分 | 5.00~100+ |
| TTG | 总进球 | 0/1/2/3/4/5/6/7+ | 2.50~20.00 |
| BQC | 半全场 | 半场+全场组合 | 2.00~50+ |
| 混合过关 | 多玩法串关 | 跨玩法组合 | 赔率相乘 |

## 票据分析流程

1. **提取选项**：从截图中识别每场选择的玩法+选项+赔率
2. **计算模型概率**：用v6.0模型计算每个选项的真实概率
3. **计算EV**：EV = 模型概率 × 赔率 - 1
4. **识别问题**：全部负EV? 概率过低? 串关过多?
5. **给出优化建议**：替换负EV选项为正EV选项

## 常见问题

1. **比分串关**：CRS每个选项都是负EV（竞彩抽水15-20%），4串1后负EV叠加到-70%
2. **SPF主胜低赔**：1.26赔率的主胜EV通常是-19%（模型64% vs 赔率隐含70%）
3. **2串1过高赔率**：25.00赔率的组合命中率通常<1%
4. **全负edge**：当所有选项都是负EV时，均值-方差优化(λ=5)最小化损失

## 优化策略

- **用RQSPF替代SPF**：让球后赔率更合理，EV可能转正
- **用均值-方差分配**：edge最高的选项分配最多资金
- **限制串关数**：2串1最优，3串1次之，4串1几乎必亏
- **跳过低置信度场次**：模型置信度<50%时不下注

## Pitfall

1. **竞彩抽水15-20%**：比国际博彩公司(5-6%)高很多，负EV是常态
2. **比分投注本质**：CRS每个选项都是负EV，只适合娱乐
3. **混合过关组合数**：2选1×2选1=4注，3选1×3选1=9注，指数增长
4. **最高奖金≠期望收益**：最高奖金是全中场景，期望收益要考虑概率
