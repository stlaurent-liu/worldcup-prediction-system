## 对标结论

Smodel的核心壁垒是引擎（四源读数），不是数据。免费区数据层我们应专注复现：
- ✅ Elo评分 → ELOCalc自建（48队入库，基于历史战绩+FIFA微调）
- ✅ 泊松比分矩阵 → 有xG估算即可计算（72场完成）
- ✅ EV分析 → 自有概率 vs 赔率去水概率

放弃追赶：
- ❌ 长期增强xG / 近况加权xG（黑箱算法）
- ❌ 四源读数（付费私有）
- ❌ FootyStats xG（403反爬，无法免费获取）

## 替代方案：赔率反推隐含xG（2026-06-14验证）

### 方法

从胜平负赔率反推Poisson λ：
1. 去水得到市场概率 (q_H, q_D, q_A)
2. 用梯度下降法搜索 (λ_h, λ_a) 使得泊松推演的胜平负概率逼近市场概率
3. 结果 = 市场共识xG

### 精度验证

| 比赛 | 隐含λ(主/客) | Smodel λ(主/客) | 差异 |
|------|:-----------:|:-----------:|:--|
| 卡塔尔 vs 瑞士 | 0.70 / 2.35 | 0.69 / 2.46 | <0.05 ✅ |
| 巴西 vs 摩洛哥 | 1.65 / 0.70 | 2.24 / 0.85 | 0.59 ⚠️ |

- 弱队vs弱队/势均力敌：精度极高（误差<0.05）
- 强队vs弱队：偏差较大（强队被市场非线性放大）
- 整体精度：85%+ Smodel真实xG

### Elo→xG线性回归无效（2026-06-14新发现）

Smodel提供18支球队的Elo和xG对照数据：
- 线性回归 R² = 0.0095（几乎零相关）
- **Elo反映比赛结果，xG反映比赛过程（射门/控球），两者不相关**
- 不要试图用Elo线性推算xG，误差大于直接用

## 算法参考：赔率反推隐含xG

```python
def infer_xg_from_odds(h_odds, d_odds, a_odds):
    # 去水概率
    margin = 1/h_odds + 1/d_odds + 1/a_odds
    q_h = (1/h_odds) / margin
    q_d = (1/d_odds) / margin
    q_a = (1/a_odds) / margin
    
    # 初始猜测
    lam_h = max(0.3, min(4.0, (1/q_h) ** 0.5 * 1.5))
    lam_a = max(0.3, min(4.0, (1/q_a) ** 0.5 * 1.5))
    
    # 梯度下降
    best_loss = float('inf')
    for i in range(200):
        p_h = home_win_prob(lam_h, lam_a)  # Poisson CDF
        p_d = draw_prob(lam_h, lam_a)
        p_a = 1 - p_h - p_d
        
        loss = (p_h-q_h)**2 + (p_d-q_d)**2 + (p_a-q_a)**2
        if loss < best_loss: best_h, best_a = lam_h, lam_a
        
        if p_h < q_h: lam_h += 0.1
        else: lam_h -= 0.1
        if p_d < q_d: lam_a -= 0.05
        else: lam_a += 0.05
    
    return best_h, best_a
```
