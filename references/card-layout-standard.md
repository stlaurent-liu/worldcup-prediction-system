# 竞彩报告卡片式排版标准（2026-06-16 南山确认）

## 触发场景
当生成竞彩足球分析、比分预测、投注建议等报告时，使用此排版标准。

## 核心规则

### 符号体系（只有一套，绝不混用）
| 符号 | 用途 |
|:|:|
| `═════` (74个=) | 大区块边界（标题区/来源区） |
| `┌─┐` `└─┘` | 卡片框（每场比赛一个完整卡片） |
| `├─┤` | 卡片框顶标题后的分割线 |
| `│` | 卡片左右边框 |
| `───` (74个-) | 框内栏目标内部分隔 |

**绝对禁止混用** `+---+` 或 `+-+-+-+-+-+-+-+-`。混用会让排版看起来"乱糟糟"。

### 投注建议卡片
投注建议放在最后，用嵌套框区分方案A/B：
```
┌──────────────────────────────────────────────────────────────────────────────┐
│  ③ 投注建议                                                                  │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────────┐      │
│  │  方案A - 2串1 (推荐)                                            │      │
│  ├────────────────────────────────────────────────────────────────────┤      │
│  │  乌拉圭客胜(1.31) × 伊朗主胜(1.56) = 串关2.04                    │      │
│  │  投入100元 → 命中回报204元 → 利润104元                           │      │
│  └────────────────────────────────────────────────────────────────────┘      │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────────┐      │
│  │  方案B - 比分单注 (高风险高回报)                                │      │
│  ├────────────────────────────────────────────────────────────────────┤      │
│  │  比分赔率约8-12倍 → 投入20元命中可获160-240元                     │      │
│  └────────────────────────────────────────────────────────────────────┘      │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

### Emoji栏目标识
| Emoji | 栏目名 | 用途 |
|:--|:--|:--|
| 📋 | 基础数据 | SPF赔率、泊松λ、融合概率 |
| 🧮 | 比分概率 | TOP5-10比分及其概率 |
| 💰 | EV分析 | 选项、赔率、模型概率、EV |
| 🎯 | 推荐比分 | 首选/次选比分 |
| ★ | 综合判断 | 整体策略建议 |

### 关键标记
| 标记 | 含义 |
|:--|:--|
| ✅ | 正EV / 已完成 / 命中 |
| ⚠ | 风险提示 / 分歧 |
| ❌ | 负EV / 不可行 / 未命中 |
