## 数据库

```sql
-- wc2026_schedule 表结构
CREATE TABLE wc2026_schedule (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    group_name TEXT,         -- 组别 A-L
    home_zh TEXT,            -- 主队中文
    away_zh TEXT,            -- 客队中文
    home_en TEXT,            -- 主队英文
    away_en TEXT,            -- 客队英文
    match_date TEXT,         -- 日期 2026-06-12
    match_time TEXT,         -- 时间 03:00
    kickoff_cst TEXT         -- 北京时间 ISO格式
);

-- 查询示例
SELECT group_name, home_zh, 'vs', away_zh, match_date, match_time
FROM wc2026_schedule
WHERE match_date >= '2026-06-14' AND match_date <= '2026-06-15'
ORDER BY match_date, match_time;
```
