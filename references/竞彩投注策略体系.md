# 世界杯竞彩投注策略体系

> 基于实战复盘 + 数据验证 + 标准投注理论
> 2026-06-13

## 一、核心原则
═══════════════════════════════════════════════════════════════

### 1.1 资金管理
┌───────────────────────────────────────────────┐
│ 单日总预算     ≤ 日收入的 2-5%                │
│ 单票上限       ≤ 总预算的 40%                 │
│ 保本覆盖率     ≥ 60%（中任何一张能回本）       │
│ 连黑停手       3连黑后强制休息1天              │
└───────────────────────────────────────────────┘

### 1.2 玩法优先级（按胜率排序）

| 优先级 | 玩法 | 胜率 | 适合场景 |
|:------:|:----:|:----:|:--------:|
| ① | 让球胜平负 | ★★★★ | 实力差≤15分 |
| ② | 胜平负单关 | ★★★☆ | 实力差≥15分 |
| ③ | 进球数 | ★★★ | 实力接近/小球格局 |
| ④ | 比分单关 | ★★☆ | 覆盖3-4个比分 |
| ⑤ | 2串1 | ★★ | 两个方向都高置信 |
| ⑥ | 半全场 | ★ | 仅限强弱分明场次 |

## 二、三维度决策模型
═══════════════════════════════════════════════════════════════

每场比赛从三个维度评分（1-5分），总分≥12分才出手：

### 维度1: 基本面（实力分差值）
┌───────────────────────────────────────────────┐
│ FootyStats实力差 | 得分 | 说明               │
│───────────────────────────────────────────────│
│ ≥25分            | 5分  | 碾压局,方向明确     │
│ 15-24分          | 4分  | 有明显实力优势     │
│ 8-14分           | 3分  | 实力占优但非碾压   │
│ 3-7分            | 2分  | 接近,主客场定胜负  │
│ ≤2分             | 1分  | 五五开,跳过        │
└───────────────────────────────────────────────┘

### 维度2: 赔率价值（EV信号）
┌───────────────────────────────────────────────┐
│ EV值     | 得分 | 说明                       │
│──────────┼──────┼────────────────────────────│
│ >+5%     | 5分  | 显著正EV,强信号            │
│ +2~+5%   | 4分  | 正EV,可出手               │
│ -2~+2%   | 3分  | 中性,看其他维度            │
│ -5~-2%   | 2分  | 负EV,谨慎                │
│ <-5%     | 1分  | 显著负EV,跳过             │
└───────────────────────────────────────────────┘
计算方式: 去水概率(WhoScored) × Sporttery赔率 - 1

### 维度3: 多源一致性
┌───────────────────────────────────────────────┐
│ 条件                    | 得分 | 说明         │
│─────────────────────────┼──────┼──────────────│
│ SP+WS+B3方向完全一致    | 5分  | 市场共识     │
│ SP和WS方向一致但赔率差<0.1| 4分 | 定价合理    │
│ SP和WS方向一致但赔率差>0.1| 3分 | 有分歧      │
│ SP和WS方向不一致        | 2分  | 方向不明     │
│ 只有一个数据源          | 1分  | 不确定性高   │
└───────────────────────────────────────────────┘

## 三、比分覆盖策略（关键！）
═══════════════════════════════════════════════════════════════

比分不能只押一个方向，要按**梯度覆盖**：

### 3.1 实力差≥20分（碾压局）
│ 比分   | 覆盖比例 | 典型赔率 | 说明             │
│────────┼──────────┼──────────┼──────────────────│
│ 2-0    | 25%      | 6-8     | 最可能净胜2球    │
│ 3-0    | 20%      | 10-15   | 全面碾压         │
│ 3-1    | 20%      | 8-12    | 客队挽回一球     │
│ 4-0    | 15%      | 20-30   | 崩盘局           │
│ 2-1    | 10%      | 6-8     | 小胜但丢球       │
│ 4-1    | 10%      | 20-30   | 大胜客队进球     │
└────────────────┴──────────────────────────────┘
案例: 美国(51) vs 巴拉圭(43) → 实际4-1 ✅

### 3.2 实力差8-19分（有优势但非碾压）
│ 比分   | 覆盖比例 | 说明                     │
│────────┼──────────┼──────────────────────────│
│ 1-0    | 25%      | 小胜最可能               │
│ 2-0    | 20%      | 正常优势体现             │
│ 2-1    | 20%      | 客队有进球能力           │
│ 1-1    | 15%      | 被逼平                   │
│ 3-1    | 10%      | 攻击力爆发               │
│ 3-0    | 10%      | 防守成功                │
└────────────────┴──────────────────────────────┘

### 3.3 实力差<8分（五五开）
│ 比分   | 覆盖比例 | 说明                     │
│────────┼──────────┼──────────────────────────│
│ 1-1    | 30%      | 最可能结果               │
│ 1-0    | 20%      | 主队小胜                 │
│ 0-1    | 20%      | 客队小胜                 │
│ 0-0    | 15%      | 闷平                     │
│ 2-1    | 10%      | 大球格局                │
│ 2-0    | 5%       | 一边倒                  │
└────────────────┴──────────────────────────────┘

## 四、标准仓位分配（每场100元）
═══════════════════════════════════════════════════════════════

### 方案A: 实力碾压局（实力差≥15分）— 保本+进攻型
┌───────────────────────────────────────────────┐
│ 票1: 主胜/客胜单关    40元  → 保本线         │
│ 票2: 比分覆盖(4个方向) 40元  → 10元×4个比分  │
│ 票3: 半全场胜胜        10元  → 博高赔         │
│ 票4: 进球数            10元  → 覆盖2-3球      │
└───────────────────────────────────────────────┘
案例: 巴西(83) vs 摩洛哥(61) — 实力差22分

### 方案B: 实力接近局（实力差5-14分）— 均衡型
┌───────────────────────────────────────────────┐
│ 票1: 主胜单关          30元  → 打底           │
│ 票2: 平局单关          20元  → 防冷           │
│ 票3: 比分覆盖(3个方向) 30元  → 10元×3个比分  │
│ 票4: 让球胜平负        20元  → 看盘口        │
└───────────────────────────────────────────────┘

### 方案C: 弱主vs强客局（实力差≥25分+主胜<1.70）— 客场进攻型
┌───────────────────────────────────────────────┐
│ 票1: 客队受让(+0.5)   50元  → 核心仓位        │
│ 票2: 客胜单关          20元  → 正EV追击       │
│ 票3: 客队比分覆盖      20元  → 0-1/0-2/1-2   │
│ 票4: 1-1平局           10元  → 主场扳平       │
└───────────────────────────────────────────────┘
案例: 加拿大(42) vs 波黑(72) — 受让(+0.5)✅

## 五、串关策略
═══════════════════════════════════════════════════════════════

### 5.1 2串1黄金法则
┌───────────────────────────────────────────────┐
│ "一稳一搏"原则:                                │
│ 稳: 实力差≥15分的碾压局,赔率<1.60             │
│ 搏: EV>+5%的高赔方向,赔率>2.50                │
│ 总赔率控制在 2.0-4.0 之间                     │
└───────────────────────────────────────────────┘

### 5.2 比分串关
┌───────────────────────────────────────────────┐
│ "同组同方向"原则:                              │
│ 两场都选同一个比分方向(如都是1-0/2-0)          │
│ 或一场比分×一场胜平负(混合过关)               │
│ 单注≤10元,最多2串1                             │
└───────────────────────────────────────────────┘
