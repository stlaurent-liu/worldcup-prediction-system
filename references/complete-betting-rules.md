# 竞彩足球完整玩法规则手册

> 来源：中国体彩官网(lottery.gov.cn) + 500.com官方规则
> 更新：2026-06-27  
> 体彩/外盘分界见 `sporttery-vs-overseas-rules.md`

## 六种玩法总览

| 玩法 | API代号 | 选项数 | 最高关数 | 单关支持 | 单注上限 |
|:|:|:|:|:|:|
| 胜平负 | HAD (SPF) | 3 | 8关 | ✅ 场次开放 | 20,000元 |
| 让球胜平负 | HHAD (RQSPF) | 3 | 8关 | ✅ 场次开放 | 20,000元 |
| 比分 | CRS | 31 | 4关 | ✅ 场次开放 | 20,000元 |
| 总进球 | TTG | 8 | 6关 | ✅ 场次开放 | 20,000元 |
| 半全场 | HAFU (BQC) | 9 | 4关 | ✅ 场次开放 | 20,000元 |
| 混合过关 | MIX | — | 木桶原则 | — | 20,000元 |

**单关开放规则（场次开放制）：** **全部六种玩法**均按场次开放，并非「胜平负/让球永远可单关」。2026 世界杯实测：071、070 等场次 HAD/HHAD 为 `single:0`（仅过关）；068、072 等场次 HAD 未开售、仅 HHAD 在售。

**核实方法：** `getFixedBonusV1.qry?clientCode=3001&matchId=XXX` → `value.oddsHistory.singleList`。`single:1` = 可单关；`single:0` = 仅过关（须至少 2串1）。另查 `getMatchListV1` 的 `oddsList` 确认玩法是否开售（h/d/a 非空）。

**两层陷阱：** ① 玩法未开售（oddsList 为空）② 玩法在售但仅过关（single:0）。二者不可混淆。

## HAD/HHAD（胜平负 / 让球胜平负）

- 竞猜全场90分钟（含伤停补时）结果
- 3种选项：主胜(H)、平(D)、客胜(A)
- 单关与否按场次开放；部分场次不售 HAD，只售 HHAD
- HHAD 仅整数让球（±1/±2/±3），**不等于外盘亚盘小数盘**
- 投注倍数：2-99倍

### 让球计算（官方规则）

```
主队让1球（handicap=-1）：
  胜：主队得分 - 客队得分 > 1  （如2:0, 3:1）
  平：主队得分 - 客队得分 = 1  （如1:0, 2:1, 3:2）
  负：主队得分 - 客队得分 < 1  （如0:0, 1:1, 0:1）

客队让3球（handicap=+3）：
  胜：客队得分 - 主队得分 < 3
  平：客队得分 - 主队得分 = 3
  负：客队得分 - 主队得分 > 3
```

常见让球数：0(平手), -1/+1, -2/+2, -3/+3

## CRS（比分）31种选项

```
主胜(13): 1:0, 2:0, 2:1, 3:0, 3:1, 3:2, 4:0, 4:1, 4:2, 5:0, 5:1, 5:2, 胜其它
平局(5):  0:0, 1:1, 2:2, 3:3, 平其它
客胜(13): 0:1, 0:2, 1:2, 0:3, 1:3, 2:3, 0:4, 1:4, 2:4, 0:5, 1:5, 2:5, 负其它
```

- 支持单关投注（场次开放制，见上表核实方法）
- 单关 = 浮动奖金（截止前赔率可能变动）；串关 = 固定奖金（投注时赔率锁定）
- 比分方案优先单关分散，勿为凑串强行 2串1

## TTG（总进球）8种选项

```
0球(~10%) | 1球(~20%) | 2球(~28%) | 3球(~22%)
4球(~12%) | 5球(~5%)  | 6球(~2%)  | 7+球(~1%)
```

## HAFU/BQC（半全场）9种选项

```
HH=胜胜(~35%) | HD=胜平(~5%)  | HA=胜负(~2%)
DH=平胜(~15%) | DD=平平(~15%) | DA=平负(~5%)
AH=负胜(~8%)  | AD=负平(~8%)  | AA=负负(~7%)
```

- 按**原始比分**的半场+全场胜平负判定，**不受 HHAD 让球影响**

## 混合过关核心规则

1. **同一场比赛的不同玩法不能混串**（硬性规则）
2. **不同运动项目不能混串**
3. **木桶原则**：关数上限 = 所含玩法中最低的
   - SPF(8关) + TTG(6关) → 最高6关
   - SPF(8关) + CRS(4关) → 最高4关
4. **冠军/冠亚军不能串关**

## 串关与容错（M串N）

| 类型 | 说明 | 注数 |
|:|:|:|
| 2串1 | 2场全对 | 1注 |
| 3串1 | 3场全对 | 1注 |
| 2串3 | 3场中任意2场 | 3注 |
| 3串4 | 3场全对或任意2场 | 4注 |
| 4串1 | 4场全对 | 1注 |
| 4串11 | 4场全对/3场/2场 | 11注 |

注数公式：C(N,M) = N! / (M! × (N-M)!)

## 投注金额（出票约束，易与外盘混淆）

```
每注基本金额 = 2元（官方强制，外盘无「注」概念）
票面金额 = 注数 × 2元 × 倍数
单张彩票最大投注金额 = 20000元
```

- **单式**：1 个选项 = 1 注；1 倍时票面 2 元
- **复式**：注数 = 组合数（如 2 场各选 2 项的 2串1 = 4 注 = 8 元@1 倍）
- **倍数**：官方条文 2–99 倍；App 常见 1–50 倍，以出票界面为准
- **配仓输出**：Markowitz 理想金额须经 `quantize_stake` 转为注数/倍数；禁止只写 `73.5元`

## 奖金计算

```
过关（固定奖金）：单注奖金 = 2元 × 赔率1 × 赔率2 × ... × 赔率n（投注时赔率锁定）
单关（浮动奖金）：单注奖金 = 2元 × 开奖SP（截止后官方公布，非投注时显示值）
单注最高奖金 = 500万元
单张彩票最高奖金 = 500万元
```

**任意玩法**单关均走浮动奖金；**任意玩法**过关均走固定奖金。出票前向用户说明差异。

## 自由过关

- 支持单式、复式、多倍投注
- 单票最高50倍
- 所有关次倍数须一致
- **不支持设胆功能**
