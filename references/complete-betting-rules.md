## SPF/RQSPF

- 竞猜全场90分钟（含伤停补时）结果
- 3种选项：主胜(H)、平(D)、客胜(A)
- 支持单关投注
- 投注倍数：2-99倍

### 让球计算（官方规则）

```
主队让1球（handicap=-1）：
  胜：主队得分 - 客队得分 > 1  （如2:0, 3:1）
  平：主队得分 - 客队得分 = 1  （如1:0, 2:1, 3:2）
  负：主队得分 - 客队得分 < 1  （如0:0, 1:1, 0:1）

客队让3球（handicap=+3）：
  胜：客队得分 - 主队得分 < 3
  平：客队得分 - 主队得分 = 3
  负：客队得分 - 主队得分 > 3
```

常见让球数：0(平手), -1/+1, -2/+2, -3/+3

## CRS（比分）31种选项

```
主胜(13): 1:0, 2:0, 2:1, 3:0, 3:1, 3:2, 4:0, 4:1, 4:2, 5:0, 5:1, 5:2, 胜其它
平局(5):  0:0, 1:1, 2:2, 3:3, 平其它
客胜(13): 0:1, 0:2, 1:2, 0:3, 1:3, 2:3, 0:4, 1:4, 2:4, 0:5, 1:5, 2:5, 负其它
```

## TTG（总进球）8种选项

```
0球(~10%) | 1球(~20%) | 2球(~28%) | 3球(~22%)
4球(~12%) | 5球(~5%)  | 6球(~2%)  | 7+球(~1%)
```

## BQC（半全场）9种选项

```
HH=胜胜(~35%) | HD=胜平(~5%)  | HA=胜负(~2%)
DH=平胜(~15%) | DD=平平(~15%) | DA=平负(~5%)
AH=负胜(~8%)  | AD=负平(~8%)  | AA=负负(~7%)
```

## 混合过关核心规则

1. **同一场比赛的不同玩法不能混串**（硬性规则）
2. **不同运动项目不能混串**
3. **木桶原则**：关数上限 = 所含玩法中最低的
   - SPF(8关) + TTG(6关) → 最高6关
   - SPF(8关) + CRS(4关) → 最高4关
4. **冠军/冠亚军不能串关**

## 串关与容错（M串N）

| 类型 | 说明 | 注数 |
|:|:|:|
| 2串1 | 2场全对 | 1注 |
| 3串1 | 3场全对 | 1注 |
| 2串3 | 3场中任意2场 | 3注 |
| 3串4 | 3场全对或任意2场 | 4注 |
| 4串1 | 4场全对 | 1注 |
| 4串11 | 4场全对/3场/2场 | 11注 |

注数公式：C(N,M) = N! / (M! × (N-M)!)

## 奖金计算

```
单注奖金 = 2元 × 赔率1 × 赔率2 × ... × 赔率n
单注最高奖金 = 500万元
单张彩票最高奖金 = 500万元
```

## 自由过关

- 支持单式、复式、多倍投注
- 单票最高50倍
- 所有关次倍数须一致
- **不支持设胆功能**
