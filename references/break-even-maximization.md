# 保本收益最大化投注策略

## 核心原理

给定 `n` 个互斥结果，赔率分别为 `d₁, d₂, ..., dₙ`，预算 `B`。
需要分配 `x₁, x₂, ..., xₙ` 使得**任何命中的结果至少回收 B**，同时**剩余预算投最高赔率选项博超额收益**。

## 标准解法（2结果+1高赔）

```
保本条件: x₁·d₁ ≥ B 且 x₂·d₂ ≥ B
代入: x₁ + x₂ = B_base（最小化B_base）
解得: x₁ = B/d₁, x₂ = B/d₂, B_base = B/d₁ + B/d₂
剩余: B_extra = B - B_base
```

## 实战验证: 美国vs巴拉圭 (2026-06-13)

SP赔率: H=1.87 D=3.02 A=3.80, 预算100元

```
保本主胜线: 100/1.87 = 53.48元 → 投54元
保本平局线: 100/3.02 = 33.11元 → 投33元
剩余: 13元 → 投1-1比分@5.45
```

| 中奖场景 | 概率 | 回收 | 盈亏 |
|:--------:|:----:|:----:|:----:|
| 主胜 | 47.4% | 101元 | +1元 |
| 1-1平局 | ~10% | 171元 | +71元 |
| 其他平局 | ~19% | 100元 | 保本 |
| 客胜 | 23.3% | 0元 | -100元 |

**验证结果**: 实际美国 4-1 巴拉圭（主胜命中）
- 主胜54元 @1.87 → 100.98元
- 回收: 100.98元 ✅ 保本成功
- 保本概率: 76.7%

## 适用条件

保本总投入 < B（即 1/d₁ + 1/d₂ < 1。当两个结果去水概率之和<100%时适用）

## 通用模板

给定预算B和H/D/A赔率:

1. 计算 `x_h = round(B / d_h)`，`x_d = round(B / d_d)`
2. 如 `x_h + x_d < B`: 剩余投最高赔率比分
3. 如 `x_h + x_d >= B`: 只覆盖主胜，剩余投比分或留着
